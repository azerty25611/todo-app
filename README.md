이 프로젝트는 간단한 TODO 리스트 웹 애플리케이션입니다.  
JavaScript와 LocalStorage를 사용하여 할 일을 추가하고, 완료 처리하며, 삭제할 수 있습니다.

## 🌟 주요 기능

- ✅ 할 일 추가
- ✅ 클릭하여 완료 표시
- ✅ 우클릭으로 삭제
- ✅ LocalStorage를 활용한 데이터 저장 (새로고침해도 유지)

## 💻 사용 기술

- HTML
- CSS
- JavaScript
- LocalStorage API
- Git, GitHub

## 🧪 실행 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저에서 열면 바로 실행됩니다.
